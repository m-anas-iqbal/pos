Orion - Multilingual Inventory management system is an automation software to track the inventory record, ordering, purchasing. It also maintains the account of supplier & stock level of warehouses. Getting absolute benefit in the present competitive world it is obligatory to maintain business timely, accurately, competently and according to demand of time and customers.  Task becomes easy by automated system software.So, manage business functional data with Orion Inventory management system.

Tools

Language

Click on the Language menu. Orion supports the multilanguage functionality by means of a translated xml file. It support over 100 language. Select your languege and check set as your default languege and click on save button. Restart the software.

Translate Orion binary

If your language is not listed in the language combo , then you have to do the following steps to test your translation:

  1. Rename English.xml to nativeLang.xml
  2. Open this .xml file with any source editor.(You can use Notepad ++)
  3. All the English words under the Attribute name should be translated.
  4. Put this translated file Language folder inside application directory folder.
